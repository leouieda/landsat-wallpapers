# Landsat Wallpapers

> Beautiful wallpapers made from Landsat remote-sensing data.

Uses [xlandsat](https://www.compgeolab.org/xlandsat/) and public domain data
downloaded from [EarthExplorer](https://earthexplorer.usgs.gov/).
